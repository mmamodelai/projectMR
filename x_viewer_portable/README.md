# X-Viewer Portable Package

This folder contains a complete, portable external budtender management system that can be given to clients.

## Files Included:

- **`x_viewer_portable.py`** - The main portable viewer application
- **`start_x_viewer_portable.bat`** - Auto-setup launcher for Windows
- **`CLIENT_INSTRUCTIONS.md`** - Complete instructions for clients

## How to Use:

### For Clients (Windows):
1. Double-click `start_x_viewer_portable.bat`
2. Wait for automatic setup
3. Viewer opens automatically

### For Clients (Mac/Linux):
1. Install Python 3.7+ from python.org
2. Run: `pip install supabase`
3. Run: `python x_viewer_portable.py`

## Features:
- **524 budtenders** from **112 dispensaries**
- **Live search** functionality
- **Points editing** system
- **Professional GUI** interface
- **Cloud database** connection (always up-to-date)

## Requirements:
- Python 3.7 or higher
- Internet connection
- supabase package (auto-installed)

## Data Source:
Connects to Supabase cloud database with all budtender and dispensary information.
