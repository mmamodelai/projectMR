## HOW TO EXECUTE

1. Create strain list file:
   ```
   OG Kush
   Blue Dream
   Cookies
   Sour Diesel
   ... (etc)
   ```

2. Run batch scraper:
   ```bash
   cd leafly
   python leafly_scraper.py --batch expansion_strains.txt --output expansion_data.json
   ```

3. Run import to Supabase:
   ```bash
   cd supabase-integration
   python import_leafly_to_supabase.py
   ```
